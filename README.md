# Kios Laundry

Landing page sederhana untuk jasa laundry menggunakan Next.js dan Tailwind CSS.

## Fitur
- Responsive
- Hero section
- Paket layanan laundry
- WhatsApp button
- Sticky navbar
- Animasi sederhana

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## Cara Menjalankan

Install dependency:

```bash
npm install
```

Jalankan project:

```bash
npm run dev
```

Buka di browser:

```bash
http://localhost:3000
```

## Struktur Folder

```bash
app/
components/
public/
```

## Custom

### Ganti nomor WhatsApp
Cari:

```bash
6281234567890
```

Lalu ganti dengan nomor sendiri.

### Ganti harga layanan
Edit file:

```bash
components/sections/Services.tsx
```

### Ganti metadata website
Edit:

```bash
app/layout.tsx
```

## Deploy

Project bisa deploy menggunakan Vercel.
